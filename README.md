# Odyxy Deploy Site

This folder contains the static deployable site for the Odyxy project (HTML, JS, and supporting resources).

How to use

- Open `index.html` or preview the folder in a static server to view the site.
- To publish to GitHub Pages, push this repository to GitHub and enable Pages in repository settings.

Contents

- HTML files for the site
- `resources/` with images and assets

Created and committed by automation via local `gh` CLI.
